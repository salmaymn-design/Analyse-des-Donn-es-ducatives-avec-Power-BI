# 📊 Analyse des Données Éducatives avec Power BI

## 📌 Description du projet

Ce projet a pour objectif d'analyser les performances académiques des élèves à l'aide de **Microsoft Power BI**. Il permet de transformer des données éducatives en tableaux de bord interactifs afin d'aider les responsables pédagogiques à suivre les résultats scolaires, identifier les élèves à risque et évaluer les performances des enseignants.

---

# 🎯 Objectifs

* Suivre les performances des élèves.
* Mesurer le taux de réussite.
* Identifier les élèves à risque.
* Évaluer les performances des enseignants.
* Analyser les résultats par matière, niveau scolaire et semestre.
* Fournir des recommandations basées sur les données.

---

# 🛠️ Outils utilisés

* Microsoft Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Modèle relationnel (Star Schema)

---

# 📂 Structure des données

Le projet utilise plusieurs tables :

* Students
* Teachers
* Courses
* Classes
* Grades
* Attendance
* School Years

---

# 📈 Mesures DAX principales

Les indicateurs suivants ont été créés :

* Total Students
* Average Grade
* Success Rate
* Number of At-Risk Students
* Average Attendance
* Total Teachers
* Average Teacher Performance
* Average Grade by Subject
* Absence Rate

---

# 📊 Dashboards

## 📄 Page 1 — Vue Élèves

Cette page présente une vue globale des performances des élèves.

### KPI

* Total des élèves
* Moyenne générale
* Taux de réussite
* Nombre d'élèves à risque

### Visualisations

* Répartition des élèves par niveau scolaire
* Évolution de la moyenne générale
* Répartition des élèves selon la réussite
* Scatter Chart des élèves à risque

---

## 📄 Page 2 — Vue Enseignants

Cette page permet d'analyser les performances des enseignants.

### KPI

* Nombre total d'enseignants
* Performance moyenne
* Nombre de cours enseignés

### Visualisations

* Top 5 enseignants
* Flop 5 enseignants
* Performance par département
* Performance par type de contrat

---

## 📄 Page 3 — Analyse des Matières

Cette page présente les performances académiques par matière.

### KPI

* Moyenne des notes
* Nombre de matières
* Meilleure matière
* Matière la moins performante

### Visualisations

* Moyenne par matière
* Répartition des notes
* Matières performantes (Moyenne ≥ 12)
* Évolution des résultats par semestre

---

## 📄 Page 4 — Conclusions et Recommandations

Cette page résume les principaux résultats de l'analyse.

### Principales observations

* Les matières présentant les meilleures performances sont identifiées.
* Les élèves avec un faible niveau de réussite sont détectés grâce aux données de notes et d'absences.
* Les enseignants les plus performants sont mis en évidence.
* Les différences de performance selon les niveaux scolaires sont observées.

### Recommandations

* Renforcer l'accompagnement des élèves à risque.
* Réduire l'absentéisme scolaire.
* Mettre en place un suivi régulier des matières les moins performantes.
* Valoriser les meilleures pratiques pédagogiques des enseignants les plus performants.
* Utiliser les tableaux de bord pour le suivi des décisions pédagogiques.

Grâce à ce tableau de bord, les utilisateurs peuvent :
